# Codex OpenTelemetry (Otel)

## Overview
`codex-otel` sets up and manages OpenTelemetry observability, including metrics and traces.

## Responsibility
- **Observability:** Configures OpenTelemetry providers (`otel_provider.rs`).
- **Transport:** Handles OTLP export (`otlp.rs`).

## Key Files
- **`src/lib.rs`**: Library entry point.
