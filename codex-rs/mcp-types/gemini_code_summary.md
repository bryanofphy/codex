# Codex MCP Types

## Overview
`codex-mcp-types` contains shared type definitions for the Model Context Protocol (MCP). It serves as a common dependency for both MCP servers and clients within the Codex ecosystem.

## Responsibility
- **Type Definitions:** Defines the core data structures used in MCP communication.

## Key Files
- **`src/lib.rs`**: Library entry point containing the type definitions.
